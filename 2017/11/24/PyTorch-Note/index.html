PyTorch Note

This note is to put a simple example code from official site for later reference.

Note

structures

The code is divided into 4 parts: (1) define network, (2) define loss, (3) data loading and (4) training loop

import

(1)torch.nn include layers as module. For example nn.Conv2d(3, 6, 5), nn.MaxPool2d(2, stride=2).
(2)torch.nn.functional include function.
Both torch.nn and torch.nn.functional have relu but one is a module and the other is a function.

ways to define network

Two main ways to define a network: (1) nn.Sequential() and (2) nn.Module inherited class

nn.Sequential() directly puts module as sequence for use:

nn.Module is able to re-define forward() for more flexible network as above

loss and optim

PyTorch defines loss functions here, such as nn.L1Loss, nn.MSELoss, nn.CrossEntropyLoss and nn.NLLLoss.
(the difference between nn.CrossEntropyLoss and nn.NLLLoss is the former take the last layer without softmax)

PyTorch defines optimizers here, such as optim.Adam and optim.SGD. One can also use optim.lr_scheduler to adjust lr.
example:

more different schedulers and examples can be found here in the bottom.

variables

  • The input and output of a network are variables, hence samples (inputs and labels) has to be converted to variables before sent to networks.
  • Tutorial here provides a simple example to clearly show variable and its gradients.
  • A variableâ€™s gradient has to be cleared before next iteration so optimizer.zero_grad() is needed
  • loss obtains outputs from criterion and loss.backward() calculates gradients with default values. optimizer.step() computes the updates with gradients.

dataloader

You can use dataset in three step: (1) dataset, (2) data loader (3) data transform

  • transforms.Compose() composes a list of transforms. transforms can be used with torchvision.datasets.CIFAR10, torchvision.datasets.MNIST or torchvision.datasets.ImageLoader (ImageLoader is very useful!)
  • Once datasets is set, torch.utils.data.DataLoader can be used for parallelly loading data.
  • If you have a list of image names in a txt file, you may consider to use ListDataset with pytorch/tnt.
    One might also consider to use Engine to efficiently construct a network.
